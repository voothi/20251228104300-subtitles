# Release Notes

## v1.0

**Date:** 2025-12-28

### Initial Release

This release marks the establishment of the subtitle creation workflow.

*   **Optimal Process**: Defined a comprehensive process for creating subtitles using Subtitle Edit and custom utilities.
*   **Tools Integration**: Integrated `20250228230803-whisper` utility, modified to support a "word-by-word" subtitle creation mode (`--one_word 2`), allowing for precise timing and editing.
*   **Documentation**: Added detail workflow documentation in README and RFCs.
